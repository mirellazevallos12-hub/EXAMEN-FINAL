# EXAMEN-FINAL

BusFinal es una aplicación web para la compra de boletos y la consulta de servicios desarrollada con React y Vite.

## Descripción

Esta página permite buscar destinos, seleccionar fechas y completar una compra de entradas con un flujo de formulario moderno. Está pensada para mostrar la funcionalidad de un sistema de reservas.

## Ejecutar localmente

Instala las dependencias:

```bash
npm install
```

Inicia el servidor de desarrollo:

```bash
npm start
```

Abre `http://localhost:3000/` en tu navegador para ver la aplicación.

## Scripts disponibles

- `npm start`: Inicia Vite en modo desarrollo.
- `npm run build`: Genera los archivos de producción.
- `npm run preview`: Previsualiza el build localmente.
- `npm test`: Ejecuta las pruebas con Vitest.

## Estructura principal

- `src/`: contiene componentes, páginas y estilos.
- `public/`: recursos estáticos.
- `index.html`: punto de entrada de la aplicación.
